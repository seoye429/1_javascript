-onchange: HTML 요소가 변경될때
-onclick: 사용자가 html 엘리먼트 클릭할때
-onmouseover: 사용자가 html 요소위로 마우스를 이동하는것,
-onmouseout: 사용자가 html 요소에서 마우스를 멀리 이동함
-onkeydown: 사용자가 키보드에 키를 누르는것
-onload: 브라우저가 페이지 로딩을 완료하는것







HTML 이벤트는 브라우저가 수행하는 일이거나 사용자가 수행하는 일일수 있다,
-페이지 로딩이 되였습니다
-입력필드가 변경 되였습니다
-버튼이 클릭되였습니다
